# Dash-Vendas
Dashboard de vendas

Dashboard de Análise de Vendas

Este projeto foi desenvolvido com o intuito de demonstrar meus conhecimentos práticos em **Power BI** e **Power Query**.

Funcionalidades do Dashboard

O relatório oferece uma visão detalhada do desempenho de vendas, permitindo analisar:
* **Período de Vendas:** Acompanhamento temporal dos produtos vendidos.
* **Análise Geográfica:** Identificação de quais continentes e países lideram as compras de determinados produtos.
* **Destaques de Produto:** Visualização clara de qual produto foi o mais vendido.
